What do I do?

I build dynamic websites using the front-end trinity of HTML, CSS and Javascript along with a variety of related libraries, frameworks and back-end building blocks.

With each project I challenge myself to not only solve a worthwhile problem or create value for the user, but to also learn a new library, framework or a better way of doing things.

Below is the first project that I've accomplished to date. You should revisit periodically for updates:

FarmgateJam

I wanted to analyze farmgate data provided by a local agricultural institution, but the experience was very frustrating (I want to be frustrated too).

As it turns out, I wasn't the only person who felt this way.

So, for my first project I decided to build a website that would make it easier to analyze and visualize the data.

The feedback (See Feedback on Twitter) showed that I still have room for improvement, but it's comforting to know that I accomplished my goal of designing a better user experience.
